# Changelog

## ggmemo (development version)

- Added
  [`annotate_change()`](https://lindsay-lintelman.github.io/ggmemo/reference/annotate_change.md)
  for visualizing the change between two data points with auto-computed
  deltas, formatted labels, and color-coded arrows.
- Added
  [`annotate_callout()`](https://lindsay-lintelman.github.io/ggmemo/reference/annotate_callout.md)
  for pointing at a data row with an arrow and label.
- Package skeleton and infrastructure set up.
